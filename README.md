# esp32-sprinkler-controller using esphome
Sprinkler Controller using esp32 and waveshare 8 port relay

Esp32-wroom32
Waveshare 8 port relay chassis

Pinouts and gpio are listed in the config - will add a drawing as well as parts list for the waveshare, the PSU used, the EPS used, and the epshome config


GPIO32 <=> Relay 1 Input
GPIO21 <=> Relay 2 Input
GPIO27 <=> Relay 3 Input
GPIO22 <=> Relay 4 Input
GPIO23 <=> Relay 5 Input
GPIO26 <=> Relay 6 Input
GPIO25 <=> Relay 7 Input
GPIO33 <=> Relay 8 Input
